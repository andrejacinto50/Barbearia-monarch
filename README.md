# Barbearia Monarch

Projeto demo em Next.js para usar como exemplo de portfólio para uma barbearia premium.

## Como rodar

```bash
npm install
npm run dev
```

Depois abra:

```bash
http://localhost:3000
```

## O que você pode editar rápido

- Número do WhatsApp: `app/page.js`
- Textos do site: `app/page.js`
- Cores: `app/globals.css`
- Imagem principal e galeria: `app/page.js`

## Observação

As imagens usadas estão em links externos. Se quiser, depois eu posso te montar uma versão com imagens locais dentro da pasta `public`.
